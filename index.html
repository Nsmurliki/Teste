Tipos Métodos

Em java existem varios tipos de métodos, os mais comuns e básicos são os métodos GET e SET. O método SET é um tipo de método sem retorno, onde se pode "enviar" um tipo de dado por parâmetro para a váriavel. Já o método GET é o inverso do SET, ele busca o tipo de dado que foi "setado" para a váriavel e obrigatóriamente tem retorno, pois é um método com retorno de dado. Métodos de Classes

Em um classe podemos ter um tipo de método que restringe ou deixa publico o acesso aos dados da mesma por outra classe sem precisar de um objeto. são eles, Private, public e Protected, o método private restringe o acesso aos dados da classe onde a outra classe só pode acessar utilizando um objeto.
